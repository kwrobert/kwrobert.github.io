Cool Software Projects

Bridgy

https://github.com/wagoodman/bridgy

A tool for interacting with cloud servers, fuzzy search thru dns names and quickly sshing in, sshfs mounting directories, etc. Seems pretty neat.

Pickachu

https://github.com/DougBeney/pickachu

A graphical file and color picker for vim. Seems kind of neat.

Anaconda Project

https://github.com/Anaconda-Platform/anaconda-project

Sweet tool for constructing reproducible, fully automated data analysis and computational experiments

Scratchdir

https://github.com/ahawker/scratchdir/

Pretty simple tool for creating temporary directories inside a context manager

Vim Sneak

https://github.com/justinmk/vim-sneak

Vim plugin that implements navigation commands like f and t but for 2 characters instead of one. Works across multiple lines

Vim Instant Markdown

https://github.com/suan/vim-instant-markdown

Tool to open a markdown file in a browser and automatically refresh when it gets updated. Steals focus on Linux :(

vim-peekaboo

https://github.com/junegunn/vim-peekaboo

Tool to peek at the contents of various vim registers

EBT (Enhanced Backup Tool)

https://github.com/larrabee/ebt

Tool for flexible backups with a module for many different use cases. Provided nice API for writing backup scripts and CLI/config file for command line useage

Bashful

A cool tool written in go for specifying jobs using YAML files. This could be a great starting point for my job submission tool

https://github.com/wagoodman/bashful

SageMath

Cool mathematica-like replacement using python. FOSS

mosh

A “mobile shell” remote shell protocol that makes working at a terminal with a slow/dodgy connection less laggy and unpleasant. Replaces interactive SSH. Instant keystroke response, robust to roaming. But you’ll need working UDP.

https://mosh.org
